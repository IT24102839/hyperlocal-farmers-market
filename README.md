# Hyper-Local Digital Farmer's Market

## Overview

A MERN Stack platform that connects local farmers, fishermen, and artisans with nearby customers.

## Features

- User Authentication
- Product Listings
- Interactive Maps
- QR Code Pickup
- Seller Dashboard
- Customer Dashboard
- Admin Dashboard

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- JWT Authentication
